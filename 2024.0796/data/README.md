# Real Datasets

Here are the real data sets (.R file) needed for replicating the experiments reported on in the paper  \[Fast multinomial logistic regression with group sparsity](https://doi.org/10.1287/ijoc.2024.0796) by Sheng Fu, Shixiang Li, Kai Yu, Piao Chen and  Zhisheng Ye. The source is as follows.



* The USPS handwritten digit data are primarily sourced from http://www.cad.zju.edu.cn/home/dengcai/Data/MLData.html.
* The Activity recognition data are primarily sourced from https://archive.ics.uci.edu/ml/datasets/Activity+recognition+using+wearable+physiological+measurements.
* The MADELON data are primarily sourced from https://jundongl.github.io/scikit-feature/datasets.html.
* The ISOLET data are primarily sourced from https://jundongl.github.io/scikit-feature/datasets.html.
* The credit card data are primarily sourced from https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud/data.
